# CBK DEV PANEL #

Standalone FiveM development menu with an F1 NUI, built for private/dev servers.

## Features

- F1 NUI menu
- Player god mode
- Vehicle god mode
- Noclip with on-screen controls
- Infinite stamina
- Super jump
- Invisible toggle
- Never wanted toggle
- Heal + armour
- Teleport to waypoint
- Spawn any vehicle by model name
- Repair / clean / flip / delete current vehicle
- Max upgrade current vehicle
- Engine toggle
- Give all weapons
- Refill ammo
- Remove all weapons
- No reload / infinite clip
- Weather presets
- Time presets + custom time
- Freeze time toggle
- Blackout toggle
- Live coordinates with one-click vector4 copy

## Install

1. Put the `cbk_devmenu` folder into your `resources` folder.
2. Add `ensure cbk_devmenu` to your server.cfg.
3. Restart the resource or server.
4. Press `F1`.

## Notes

- This is intentionally standalone and does not require a framework.
- Built for dev use, not public production use.
- Vehicle spawn expects GTA model names like `adder`, `sultanrs`, `police3`, `buzzard2`, `akula`, etc.
- Noclip controls while enabled:
  - `W/S`: forward/back
  - `A/D`: left/right
  - `Q/E`: up/down
  - `Shift`: faster
  - `Ctrl`: slower
